# Centipede Prototype

ブラウザで動作するムカデ風クリーチャーの横スクロールアクションのプロトタイプです。`index.html` をブラウザで開くだけで遊べます。

## 操作方法
- **左右キー / A・D**: 横移動
- **W / ↑**: ジャンプ
- **E**: 入手済みのエネルギーオーブを消費して身体を伸ばす
- **R**: リスタート

## ゲーム内容
ステージ上に落ちているエネルギーオーブを集めると、ムカデの身体が段階的に伸びて高所へ届きやすくなります。身体は柔らかくしなり、ヘビのような移動と壁づたいの動きが可能な挙動を目指した実装になっています。
